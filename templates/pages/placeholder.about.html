About Web Medi-Screen

Everything you want to know about Web Medi-Screen

Web Medi-Screen Establishment

The Web Medi-Screen is established through a project in September 2019. It is a group project. A group of three to four student within the Cork Institute of Technology (CIT) involved building this project. This project was built by three students Rafal Rosa, Matthew Keating and Abdul Hadi Mirzada. This project was supervised by a great experienced teacher Seamus Lankford with lots of thanks from his excellent attitude, experience and supervision.

About Web Medi-Screen

The Web Medi-Screen contains a lot of information about the different problems such as Cancer, Diabetes and Heart diseases as well as lots of related pictures.

The home page (Web Medi-Screen)

The home page contains an attraction slider, gallery some informative texts, photos, footer(contains internal and external links to Web Medi-Screen and other websites) and links to sign up or login with other applications such as Facebook and Twitter, Email and telephone number. Home page was built as one of the responsive pages

The User Profiling page

TheUser Profiling page enables insurance professionals and doctors to check reports establishing the level of risk for all categories within the website. This page can provide professionals with positive and negative indication on Diabetes, Breast Cancer and Heart Diseases reports if any available.

User Profiling shows the average age patients are suffering with these problems as well as the average BMI people have.

The Cancer page

The Cancer page contains information about Cancer, types of Cancer, Cancer spreads and internal links within the website.

The Diabetes page

This page contains information about what is Diabetes, before you got Diabetes, types of Diabetes and link for more information about Diabetes.

The Heart diseases page

The page contains massive amount of information about Heart diseases, Symptoms, Causes, Treatment, Prevention, Statistics and external links to some other websites. A great facility for going up and down as all the links at the top left side jumps straight to the topic and the "Top" red button at the right bottom of the page with jump back to the top.

Register

The Register page is going to ask the medical and insurance professionals to Register by providing username, first name, last name, email, selection of their categories from drop down menu and creating password for the login in to the system. Navigation to home page or login page is facility from this page. Register page was built responsive

Login page

The login page is provided with login or submit query and a facilities to Navigate to register and home page. Login page was built responsive
